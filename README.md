# Introdução

## Executar os testes do back-end

#### Acessa a subpasta do backend e executa o seguinte comando:

````
mvn test
````

## Executar o front-end
#### Acessa a subpasta do frontend e executa o seguinte comando:

````
npm install

yarn start
````
## Tecnologia utilizadas
#### Java SpringBoot
#### ReactJS
#### PostgreSQL
#### Heroku
